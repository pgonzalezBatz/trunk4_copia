# GIP (Gestion Integral de Proveedores)
## 1. Objetivo:
Gestión básica de proveedores
## 2. Descripción
La aplicación contiene una busqueda de proveedores accesible para todo tipo de usuarios.
El role de edición posibilita añadir y editar proveedores. El sistema mantiene sincronizados las bases de datos de SAB, Xbat, RH, Navision y grisoft
Existe el rol para mantenimiento de telefonos directos de proveedores.
Existe el rol para importar proveedores potenciales de Xbat.
Existe el rol para añadir archivos adjuntos a los proveedores.
Existe el rol para gestionar proveedores globales.
### Tecnología:
VB.NET, MVC
## 4. Instalación / Configuración

### Instrucciones paso a paso para que la app funcione en local
