Cheque gourmet

La aplicación cheque gourmet, ha sido diseñada para que el usuario pueda hacer un seguimiento de los talonarios para la comida que ha consumido. En la primera página se muestran los datos del mes actual. Tambien podemos ver el histórico de todos los talonarios comprados.

Cada trabajador podra seleccionar un máximo de un talonario por mes y grupo. Los talonarios son los siguientes:

  • Grupo S
    1. Batz para GASKA
    2. Batz para restaurante
  • Grupo C
    1. Personal para GASKA
    2. Personal para restaurante
