# Cheque Guardería
## 1. Objetivo:
Ofrecer una herramienta para hacer las solicitudes de cheuqe guardería para trabajadores con hijos menores de 3 años 
y abilitar las exportación de los datos a Epsilon.
## 2. Descripción

### Tecnología:
VB.NET, MVC

## 4. Instalación / Configuración
### Instrucciones paso a paso para que la app funcione en local
Hacer checkout del proyecto y compilarlo.
### Conexiones a las bases de Datos
Base de datos Oracle y Microsoft SQL. 
#### Oracle
Esquemas Gestionhoras, sab e izaro
#### Microsoft SQL Server
database bd_epsilon en MSSQLINS1
